# Kanban

This directory contains the Markdown Kanban Roadmap and related task detail files.

## Documents

### TASKS.md

The active project task board.

### tasks/

Task detail files.

## Rules

- Keep the board structure explicit.
- Keep task identifiers stable.
- Do not store implementation code here.
- TASKS.md is the source of truth.
- Task status is determined by the column.
- Do not add unsupported properties.
- Detailed task information belongs in tasks/T-XXX.md.
- Follow ../../00-META/skills/kanban/SKILL.md
