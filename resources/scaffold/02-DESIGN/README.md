# Design

This directory contains the project design system and brand material.

It covers visual identity, tokens, typography, UI references and design assets.

## Documents

### Brand

Directory: brand/

Brand assets and subfolders.

### System

Directory: system/

Design system documentation and structure.

### Tokens

Directory: tokens/

Design token documentation.

### Typography

Directory: typography/

Typography guidance and assets.

### UI

Directory: ui/

UI references, flows, screens and component material.

## Rule

- Keep design material separated from implementation code.
- Do not store product requirements here.
- Do not store runtime logic here.
