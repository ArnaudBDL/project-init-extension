# Product

This directory contains product-level documentation.

It includes features, kanban, roadmap, specs and UX flows.

## Documents

### Features

Directory: features/

Feature documentation.

### Kanban

Directory: kanban/

Kanban board and task detail documents.

### Roadmap

Directory: roadmap/

Roadmap documents.

### Specs

Directory: specs/

Product specification documents.

### UX Flows

Directory: ux-flows/

User experience flow documents.

## Rules

- Keep product documents separate from design and implementation code.
- Do not store runtime logic here.
