# Foundation

This directory contains the project foundation materials.

The foundation captures raw inputs, benchmarks, inspirations and research
materials that support the rest of the scaffold.

## Documents

### Foundation Assets

Directory: assets-raw/

Raw, unprocessed source assets.

### Foundation Benchmarks/

Directory: benchmarks/

Reference comparisons and evaluation material.

### Foundation Inspirations/

Directory: inspirations/

Visual, editorial or product inspiration material.

### Foundation Research/

Directory: research/

Project research notes and source analysis.

## Rules

- Keep foundation material factual and traceable.
- Do not move implementation code here.
- Do not treat inspiration material as a decision.
- Do not treat benchmark material as a requirement.
